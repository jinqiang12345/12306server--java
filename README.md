# 12306server--java
simple java spring project for react-12306

#### 这是react-12306后台的java版，使用了spring boot

#### 运行项目，可以单独测试
* clone
* 在编译器打开
* 准备mysql数据库，新建数据库java，注意排序方式选择utf-8
* run
* 打开 http://localhost:8080/swagger-ui.html#/ 可以单独测试
